
## Ignews
- Criação de uma pagina de assinatura de conteúdos.

**Rodar o projeto**

Ao clonar o projeto abra o terminal e execute os seguintes comandos:

`npm install`

`npm run dev`

**Funcionalidades**

- Assinatura de conteúdo mensal;
- Autenticação com Github;
- Entre outros.

**O que foi utilizado no desenvolvimento deste projeto?**

- Next.js
- Typescript
- CSS Modules
- API Routes
- SSG (Static Site Generation)
- SSR (Server Side Rendering)
- Next Auth com Github
- Stripe (Ferramenta de Pagamentos)
- FaunaDB
